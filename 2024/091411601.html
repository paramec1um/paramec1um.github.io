最优化问题与数学规划

数学规划
数模
发布日期:   2024-09-14

最优化问题总结

线性规划算法原理

线性规划问题的规范形式

线性规划问题普遍可以转化为如下的形式：
$$
\text{maxmize: } z=\sum_{i=1}^{m} c_ix_i=CX\\
s.t: \begin {cases}
AX=b\\
X\ge0
\end {cases}
\\
A=\begin{bmatrix}
a_{11}&a_{12}&{\cdots}&a_{1m}\\
a_{21}&a{22}&{\cdots}&a_{2m}\\
{\vdots}&{\vdots}&{\ddots}&{\vdots}&\\
a_{m1}&a_{m2}&{\cdots}&a_{mm}
\end{bmatrix}
$$

线性规划的标准形式规定：

  • 必须为最大化问题
  • 约束必须为等式
  • 决策变量必须为非负

若问题需要求解最小值，则可以通过取负或者对偶转化为求最大值

单纯形法

单纯形法主要分为如下几个步骤：

Step 0: 规范化问题，写出单纯形矩阵
Step 1: 选择初始基变量
Step 2: 判断是否为最优解；是则结束；否则继续
Step 3: 转移至下一组解，重复第二步

规范化

由于单纯形的规范形式要求不能出现不等式约束

先引入松弛变量进行松弛化，把约束不等式转化为等式，松弛变量的个数与约束方程数量一致

由于单纯形只能解最大化问题，对于最小化问题要适当转化。

转化最大化与最小化可以有取负和对偶两种操作

  1. 取负

    取负操作要把目标函数和约束方程的所有系数都变成负数，从而改变优化方向

    取负后的单纯形求解方法会有所不同，在下文会有对比

  2. 对偶

    基于对偶原则：对于问题 $p$ 描述为

$$
\text{maxmize }Z=CX
\\
s.t.\begin{cases}
AX= b\\
X\ge 0
\end{cases}
$$
​ 其对偶的问题 $D$ 可描述为
$$
\text{minmize }W=b^TY
\\
s.t.\begin{cases}
A^TY= C^T\\
Y\ge 0
\end{cases}
$$
​ 且原问题与对偶问题同时有最优解，两组最优解满足 $CX_0=b^TY_0$

​ 可以通过这个性质转化最大化与最小化问题

由于 $n$ 个决策变量，$m$ 个约束的线性规划问题的时间复杂度为 $O(n^3m^{\frac{1}{n-1}})$ 基本与 $m$ 无关

所以当 $n\ge m$ 时选择用对偶进行规范化，因为对偶操作会交换 $n,m$ ，使得总体算法时间减少

当 $n<m$ 时，选择取负，不能增加时间复杂度

单纯形矩阵

得到规范的线性规划方程后，就可以写出单纯形矩阵了，规范形式如下：

目标函数系数 $C_1$ 目标函数系数 $C_2$ $\cdots$ 目标函数系数 $C_n$ 目标值 $-Z$
约束方程系数 $a_{11}$ 约束方程系数 $a_{12}$ $\cdots$ 约束方程系数 $a_{m1}$ 方程常数 $b_1$
约束方程系数 $a_{12}$ 约束方程系数 $a_{22}$ $\cdots$ 约束方程系数 $a_{m2}$ 方程常数 $b_2$
$\vdots$ $\vdots$ $\ddots$ $\vdots$ $\vdots$
约束方程系数 $a_{m1}$ 约束方程系数 $a_{m2}$ $\cdots$ 约束方程系数 $a_{mn}$ 方程常数 $b_m$

注意：有些变量没有出现在方程中，在表中的系数填 $0$

最优解

在满足规范形的单纯形中，$C$ 值全部非正时得到最优解，意味着不论增大那个变量，目标值都不可能增加

选择替换位置

在满足规范形的单纯形中，替入变量一般选择 $C$ 值最大的

替出变量一般有两种选择方法：

  1. 最速下降法：最常用，选择 $C/a$ 比率最小的进行替出
  2. 最小引索法：直接选择引索最小的进行替出

两种选法效果相差不大

转轴操作

转轴操作也就是单纯形转移至下一个解的操作

对于选中的替入 $e$ 和替出 $l$

先把 $a_{le}$ 在 $l$ 行上归一化，即 $l$ 行除 $a_{le}$

再用高斯消元使得第 $e$ 列上，除了第 $l$ 个元素以外，其他位置为 $0$

最后交换两个变量的标记

若采用的是基变量+非基变量的矩阵，则最后一步可以省略，因为高斯消元后，原本的非基变量 $C$ 值为 $0$ 不参与后续其他的非基变量操作；而原本基变量 $C$ 值不可能为 $0$ 所以会参与到后续非基变量操作中

但是若为了省空间而只采用非基变量的矩阵，就要进行最后一步，在计算上等价于 $a_{le}=\frac{1}{a_{le}},C_e=-C_e$ 其他位置利用高斯消元计算。但是在数学意义上，因为矩阵只维护非基变量，在转轴完成后，两个变量的意义发生变化，所以要进行交换，确保矩阵当中只有非基变量

细化单纯形求解过程

正常单纯形 取负后单纯形
Step 1 选择基变量 相同
Step 2 判断 $C$ 行是否全部非正；是则最优 判断 $b$ 列是否全部非负；是则最优
Step 3 选择 $C$ 值非负且最大的列 $e$ 作为替入 选择 $b$ 值非正且最小的行 $l$ 作为换出
Step 4 遍历第 $e$ 列，选择 $C$ 值与 $a$ 值比率最小的行 $l$ 作为替出 遍历第 $l$ 行，选择 $b$ 值与 $a$ 值比率最小的列 $e$ 作为替入
Step 5 转轴，跳转到第 $2$ 步 相同

//m行，n列单纯形矩阵
//id[i]记录第i个决策变量对应的方案
//此处使用的是最速下降法来确定进基

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
const int N=1007;
const int M=10007;
const double INF=1e15;
int n,m;//m行，n列
double a[M][N],b[M];
int id[N+M];
void pivot(int l,int e){//转轴
	double k=0;//行列倍增变换系数，若直接使用行内的数值，此数值会随着元素更新而改变

	//注意：a矩阵只维护非基变量，所以转轴时，还要维护列变换的操作
	//实际上就是第le位多除一次自己，第e列除了l行的每一行多减一次自己
	//第l行除a[l][e]
	swap(id[e],id[n+l]);
	k=1/a[l][e];
	a[l][e]=1;
	for(int i=1;i<=n;++i){
        //if(i==e)continue;  //使用增广矩阵的则去掉注释
		a[l][i]*=k;//多除一次自己
	}
	b[l]*=k;

	//e列对应的所有行，除了l行都变为0
	for(int i=0;i<=m;++i){
		if(i==l)continue;
		k=a[i][e];a[i][e]=0;//0多减一次自己
		for(int j=1;j<=n;++j){
            //if(j==e)continue;  //使用增广矩阵的则去掉注释
			a[i][j]-=k*a[l][j];
		}
		b[i]-=k*b[l];
	}

	//输出中间结果 
    // for(int i=0;i<=m;++i){
	// 	for(int j=1;j<=n;++j){
	// 		cout<<a[i][j]<<' ';
	// 	}
	// 	cout<<b[i]<<'\n';
	// }
	// cout<<'\n';
}
bool simplex(){//单纯形法，确定l，e
	while(1){
		int e=0,l=0;
		for(int i=1;i<=n;++i){//确定列
			if(a[0][i]>a[0][e])e=i;//寻找最大的,大于0的系数c，f=0时，c[f]=0
		}
		if(!e)return 1;
		double mn=INF;
		for(int i=1;i<=m;++i){//确定行
			if(a[i][e]>0&&mn>b[i]/a[i][e]){
				mn=b[i]/a[i][e];
				l=i;
			}
		}
		if(!l)return 0;//w
		pivot(l,e);
	}	
}
void solve(){
	cin>>n>>m;
	for(int i=n+1;i<=m+n;++i)id[i]=i-n;
	for(int i=1;i<=n;++i){
		cin>>a[0][i];
	}
	for(int i=1;i<=m;++i){
		int s,t;
		cin>>s>>t>>b[i];
		for(int j=s;j<=t;++j)a[i][j]=1;
	}

    // 输出中间结果 
	// for(int i=n+1;i<=m+n;++i){

	// }
	// for(int i=0;i<=m;++i){

	// 		cout<<a[i][j]<<' ';
	// 	}
	// 	cout<<b[i]<<'\n';
	// }
	// cout<<'\n';

	if(simplex()){
		cout<<(long long)-b[0]<<'\n';//最优解
		// for(int i=1;i<=n+m;++i){
		// 	cout<<i<<": "<<b[id[i]]<<'\n';//最优解方案
		// }
	}
}

	ios::sync_with_stdio(0);cin.tie(0);cout.tie(0);
    fstream in("in.txt",ios::in);cin.rdbuf(in.rdbuf());
	int T=1;//cin>>T;
	while(T--)solve();

python，使用增广矩阵

import pandas as pd
import numpy as np

def pivot(d,n):
    c=list(d[0][:-2]) # 取出目标函数系数
    e=c.index(max(c)) # 最大目标函数系数的下标
    t=[]
    for i in range(n): # 第0行到第n行
        if d[i][e] == 0: # 插入0
            t.append(0.)
        else:
            t.append(d[i][-1]/d[i][e]) # 插入常数项除当前项数，准备取最小、
    l=t.index(min([x for x in t[1:] if x != 0])) # 取出非0的最小的数的下标
    s[l-1]=e # 交换
    r=d[l][e]
    d[l]/=r # 第l行除r
    for i in [x for x in range(n) if x != l]: # 其他行减去第l行的r倍
        r = d[i][e]
        d[i] -= r * d[l]

def simplex1(d,n):
    f=1
    while f:
        if max(list(d[0][:-1]))<=0:
            f=0
        else:
            pivot(d,n)

def solve(d,m):
    for i in range(m - 1):
        if i in s:
            print("x" + str(i) + "=%.2f" % d[s.index(i) + 1][-1])
        else:
            print("x" + str(i) + "=0.00")
    print("objective is %.2f" % (-d[0][-1]))

# 数据输入
d = np.array([[5, 1, 0, 0, 0], [1, 1, 1, 0, 5], [2, 1 / 2, 0, 1, 8]])
(n,m) = d.shape
s = list(range(m - n, m - 1))  # 基变量列表,初始为所有的松弛变量

# 解决问题
simplex1(d, n)
solve(d, m)

流程图

算法详细步骤（三线表版）

% 宏包
\usepackage[linesnumbered,ruled,vlined]{algorithm2e}

\begin{algorithm}[]	
	\let\oldnl\nl% Store \nl in \oldnl
	\newcommand{\nonl}{\renewcommand{\nl}{\let\nl\oldnl}}% 去掉行号
	\caption{单纯形算法求解步骤}
	\KwIn {单纯形矩阵}
	\KwOut {目标函数最优解}
	
	\nonl\textbf{Step 1:} 确定初始可行解和初始基的可行解\\
		
	\nonl\textbf{Step 2:} 检查当前表中非基变量的目标函数系数是否全部非正；若是则得到最优解，算法结束；否则进行下一步\\
		
	\nonl\textbf{Step 3:} 若单纯形表的前 $m$ 列构成单位矩阵，但在 $j=m+1...n$ 列中有某个对应 $x_k$ 的列向量 $P_k\le 0$ 停止计算，无可行解\\
		
	\nonl\textbf{Step 4:} 挑选目标函数对应行中系数最大的非基变量作为进基变量 $x_l$ ，按如下规则计算，可确定 $x_e$ 为出基变量，转下一步:
		\begin{equation}
			\theta = min\{\frac{b_i}{a_{ie}}|a_{ie}>0\}=\frac{b_l}{a_{le}}
		\end{equation}
	其中 $b_i$ 是规范型规划的常数项，$a_{ie}$ 即为在第 $i$ 个约束中变量 $x_e$ 的系数\\
		
	\nonl\textbf{Step 5:} 进行高斯消元，令列向量 $P_e$ 中除了第 $l$ 个元素为 $1$ 以外，其他全部变为 $0$\\
		
	\nonl\textbf{Step 6:} 重复步骤 $2~5$\\
\end{algorithm}
